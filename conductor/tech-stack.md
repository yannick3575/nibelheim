# Technology Stack

## Frontend
- **Framework:** Next.js 15 (App Router)
- **Library:** React 19
- **Styling:** Tailwind CSS 4
- **UI Components:** Radix UI primitives
- **Icons:** Lucide React
- **Data Visualization:** Recharts

## Backend & Database
- **Platform:** Supabase (Auth, Database, Storage)
- **Client SDKs:** `@supabase/ssr`, `@supabase/supabase-js`

## AI & Language Processing
- **LLM Provider:** Google Generative AI
- **Formatting:** `react-markdown`, `katex`, `react-katex`

## Development & Tooling
- **Language:** TypeScript
- **Testing:** Vitest, Testing Library (React, DOM), Happy DOM
- **Validation:** Zod
- **Build Tool:** Next.js built-in (SWC)
